Types of AI by Learning Style

Learning style = how it learns from information

This lens answers: “What kind of feedback does the model get while learning?”

Supervised learning

Learns from examples with the correct answers.

Example: photos labeled “cat” / “not cat”, email labeled “spam” / “not spam”.
Clear targets Common in practice

Unsupervised learning

Learns structure without labeled answers.

Example: grouping songs by similarity, finding “clusters” of customers, spotting unusual behavior.
Discover patterns No labels needed

Reinforcement learning (RL)

Learns by trial-and-error: actions → rewards/penalties.

Example: game-playing agents, robot control, some recommendation/decision systems.
Can be tricky Needs good rewards

How to talk about a real AI system

Try building a full description using all three lenses:

  • Capability: narrow vs general?
  • Functionality: reactive vs limited memory?
  • Learning style: supervised/unsupervised/RL?
Example description: “A narrow AI system trained with supervised learning to detect spam. It behaves like a reactive machine at prediction time (it looks at the email and outputs a label).”
